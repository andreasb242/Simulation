Ausführliche Beschreibung

This is the base model for both the XY and the Flow model

=== MODEL ===

The whole Simulation model uses references, for compare two objects we use always == instead of equals, because only if it's the same reference it's the same object.

Even if the name has to be unique because else it cannot be saved to an XML file
