Ausführliche Beschreibung

Allows functions to be defined in equations.

 XJep > Define("half",1,"x/2")
 XJep > half(5)

Currently the Define function is handled in the preprocessing step

 Node n = xj.parse("Define(&quot;sumToX&quot;,1,&quot;x*(x+1)/2&quot;)");
 Node preproc = xj.preprocess(n);

and preproc will be null if a Define statement is encountered. This is probably a bug.

TODO_YEP improve syntax. So can have Define("half(x)",x/2)

TODO_YEP work out how to simplify and evaluate Define statements which don't really have a value.

TODO_YEP fix parser so can do half(x) = x/2
